# ZRxJavaUtil

自己封装的一个Rxjava2 + retrofit2 网络请求框架

# 使用方法

> Step 1\. Add the JitPack repository to your build file

allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}

> Step 2\. Add the dependency

dependencies {
	        compile 'com.github.JueShouGitHub:ZRxJavaUtil:v1.0.5'
	}
